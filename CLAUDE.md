# After Inter - Development Guide

This project uses a standardized set of AI skills to ensure high-quality code, consistent UI/UX, and rigorous development practices.

## AI Skills & Commands
The following skills are available in this repository:
- `!brainstorming`: Use this before any creative work to define technical specs.
- `!writing-plans`: Use this to create structured implementation plans.
- `!receiving-code-review`: Use this when receiving feedback to ensure technical rigor.
- `!test-driven-development`: Use this to enforce TDD workflows.
- `!ui-ux-pro-max`: Use this for expert UI/UX design and CSS guidance.
- `!mem-search`: Use this to search project memory and past decisions.

## Development Workflow
1. **Brainstorm**: Define the feature and technical requirements.
2. **Plan**: Create a step-by-step implementation plan.
3. **Test**: Write failing tests first (TDD).
4. **Implement**: Build the feature according to the plan.
5. **Review**: Perform a technical review and refine.

## UI/UX Standards
- **Aesthetics**: Use rich, premium designs (vibrant colors, gradients, micro-animations).
- **Typography**: Use modern fonts (Inter, Outfit, etc.).
- **Responsiveness**: Mobile-first approach is mandatory.
- **Components**: Prefer reusable, focused components.

## Recommended Tools
- **Framer Motion**: For all animations and transitions.
- **Lucide React**: For iconography.
- **21st.dev Magic MCP**: It is highly recommended to add the 21st.dev Magic MCP to your Claude Code configuration for elite component generation.
  - Command: `claude mcp add @21st-dev/magic`
  - Requires an API key from [21st.dev](https://21st.dev/magic).
